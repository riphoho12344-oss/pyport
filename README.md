# pyport

A lightweight security tool that combines high-concurrency scanning using Python sockets with deep service identification via Nmap.

Key Features
-High-Speed Scanning: Utilizes a multi-threaded architecture where one port corresponds to one thread**, ensuring rapid reconnaissance of network assets.
- Smart Integration:** Automatically feeds discovered open ports into Nmap for detailed version detection (`-sV`).
- Customizable: Supports user-defined target IPs and port ranges.
